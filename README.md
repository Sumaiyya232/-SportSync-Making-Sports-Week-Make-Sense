🎯 SportSync – Making Sports Week Make Sense
A full-stack web application built to automate and streamline university Sports Week event management.
Created as a second-semester project for the Database Management Systems course at Institute of Space Technology (IST).

🚀 Overview
SportSync takes the stress out of Sports Week by handling everything from player registration to match scheduling and real-time score updates — all while avoiding conflicts like double-booked players, venues, or time slots.

🧩 Key Features
🔐 Secure login system for both players and admins
📝 Multi-sport player registration
📅 Auto-generated match schedules personalized for each player/team
⚠️ Conflict detection for players, venues, and timings
🛠️ Admin dashboard to manage scores and match updates in real time

🛠 Tech Stack
Frontend:	HTML, CSS, JavaScript
Backend: Node.js (Express.js)
Database:	SQL Server
Tools Used:	MSMS, Git, GitHub

📚 Course Info
Semester: 2nd
Course: Database Management Systems


💡 What I Learned
This project helped me develop skills in:

Full-stack development
Database schema design and normalization
Writing and optimizing SQL queries
Building RESTful APIs with Node.js
Designing conflict resolution logic for real-world use cases
